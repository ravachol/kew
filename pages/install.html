How to install

Linux

kew is available on most major Linux distro’s package managers.

For NixOS, there is an official package but also a Nix Flake that is tied to the latest commit.

For Artix it’s available in the artist repository.

For Fedora there is an unofficial repo.

macOS

For macOS users it’s available through Homebrew.

Windows

Windows users can download a small Windows installer.

Android

kew is available on Android as a package for the app Termux, which is a terminal emulator for Android.

Manual installation instructions

If you can’t find it on your distro, or you want the bleeding edge, follow the manual installation guide.
