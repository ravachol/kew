kew (/kjuː/, a variant of cue) is a terminal music player.

LINUX

Install through the package manager on your Linux distro, or follow the manual installation guide.

Special cases:

  • NixOS has an official package but also a Nix Flake that is tied to the latest commit.
  • On Artix it’s available in the artist repository.
  • Fedora has it on an unofficial repo.
  • kew is on Arch Linux’s official repositories, but there’s also an AUR package that tracks the latest commit called kew-git.

MACOS

For macOS users it’s available through Homebrew.

WINDOWS

Run the Windows installer, compiled and created entirely on Github.com.

ANDROID

kew is available on Android as a package for the app Termux, which is a terminal emulator for Android.
