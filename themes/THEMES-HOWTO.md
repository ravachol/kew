## How to make your own theme or edit an existing one

## Themes are overwritten as newer versions of them are available

The old one is renamed with the file extension .bak.

## Colors that can be used

- Hex colors, format:  #ffffff

- Album cover colors:
  101: Album Color 1 # This is the old color that used to be the only album color
  102: Album Color 2
  103: Album Color 3
  104: Album Color 4

- Standard Terminal ANSI Colors can also be used:
   0: Black
   1: Red
   2: Green
   3: Yellow
   4: Blue
   5: Magenta
   6: Cyan
   7: White
   8: Bright Black (Gray)
   9: Bright Red
   10: Bright Green
   11: Bright Yellow
   12: Bright Blue
   13: Bright Magenta
   14: Bright Cyan
   15: Bright White
   -1: Default terminal foreground

You can mix these three types of colors on any given layout.

## The different labels

Most theme keys correspond to a specific field in one of the views and most are self-explanatory. The "help" key corresponds to the explanations for key bindings below the log and the help view. The Core Colors aren't currently used except for "text" which is used for the labels on the help view.

## The special themes

The "albumcolors" and "onealbumcolor" themes correspond to the color mode settings "Album colors" and "One album color" respectively. The "neutral" corresponds to the Neutral color mode.

## Gradients

Gradients can't be applied to ANSI colors.
