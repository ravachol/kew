kew themes pack 3.

Included in this pack:

albumcolors - This is a special theme that decides how the album cover colors are used in album color mode. 4 available colors.
onealbumcolor - This is a special theme that emulates the old album color mode that was based on one color.
nord
kanagawa

Also, all previous themes have been improved and uses colors more restrictively

To apply, run:

kew theme <theme> # filename without the .theme part

Or cycle by pressing 't'.

These should be available under /themes here, in case you need to restore them to defaults:
