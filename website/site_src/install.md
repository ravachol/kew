## How to install

#### Linux

kew is available on most major Linux distro's package managers.

For NixOS, there is an official package but also a [Nix Flake](flake.nix) that is tied to the latest commit.

For Artix it's available in the [artist repository](https://wiki.artixlinux.org/Main/Repositories#artist_repository).

For Fedora there is an [unofficial repo](https://copr.fedorainfracloud.org/coprs/fed500/kew).

#### macOS

For macOS users it's available through [Homebrew](https://formulae.brew.sh/formula/kew).

#### Windows

Windows users can download a small [Windows installer](https://codeberg.org/ravachol/kew/releases/).

#### Android

kew is available on Android as a package for the app [Termux](https://termux.dev), which is a terminal emulator for Android.

#### Manual installation instructions

If you can't find it on your distro, or you want the bleeding edge, follow the [Manual Installation Instructions](https://codeberg.org/ravachol/kew/src/branch/main/docs/MANUAL-INSTALL-INSTRUCTIONS.md).
